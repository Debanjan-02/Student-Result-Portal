# Student Result Portal (Java + Hibernate)

A Hibernate-based CRUD application that stores and retrieves student 
data using ORM principles. Eliminates SQL boilerplate and uses entity 
mapping, session management, and transaction handling.

## Features
- Add Student Result
- Update Marks
- Fetch Student by ID
- View All Students
- Delete Record
- Hibernate annotation-based mapping

## Tech Stack
- Java
- Hibernate ORM
- MySQL

## How to Run
1. Setup MySQL DB.
2. Configure hibernate.cfg.xml.
3. Compile & run Main.java.
